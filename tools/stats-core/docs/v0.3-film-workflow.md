# v0.3 Film workflow

This version adds a dedicated Track Game tab for live/manual film review.

## Added
- Current game / half / video time context.
- FIBA shot clicker with make/miss markers and full shot metadata.
- Quick event buttons for rebounds, turnovers, steals, blocks, deflections, fouls, potential assists, advantage creation, paint touches, drive-kick creation, and screen creation.
- Quick free throw entry.
- Active lineup / lineup stint entry.
- Recent activity log and undo-last action for new entries.
- Expanded player/team reports with scoring, creation, screening, defense, shot quality, and lineup impact groups.
- JSON backup remains the source of truth; Excel export remains available.

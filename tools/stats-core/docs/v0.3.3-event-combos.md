# v0.3.3 Event combos

Changes:

- Removed `deep top 3` from the zone list and from court-click zone assignment.
- Any imported shot with `shot_zone: "deep top 3"` is normalized to `top 3`.
- Event subtype dropdowns now depend on selected event type.

Event combinations:

| event_type | allowed event_subtypes |
|---|---|
| rebound | blank, offensive, defensive |
| turnover | blank, bad_pass, lost_ball |
| foul | blank, shooting, common |
| charge_drawn | blank, offensive |
| steal | blank, live_ball |
| block | blank, rim_protection, perimeter |
| deflection | blank, live_ball, dead_ball |
| creation | blank, potential_assist, advantage_created, paint_touch_created, drive_kick_created |
| screen | blank, screen_assist, screen_opportunity, advantage_created |

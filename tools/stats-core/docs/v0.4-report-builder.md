# v0.4 Report Builder

This version keeps the v0.3.4 tracking workflow and adds a more presentable Reports tab.

## Added

- Report filters for game, half, player.
- Game report hero section with score/result context.
- Leader cards for top scorer, top creator, best screener, top rebounder, defensive activity.
- Shot-quality notes: open shot rate, bad-shot rate, assisted FG rate.
- Best zone / cold zone callouts from shot data.
- Filtered FIBA shot map inside Reports.
- Filtered shot list.
- Expanded all-player advanced table.
- Download current report HTML.

## Still to build next

- Better Games and Players pages in the mockup style.
- Opponent and strategy data model/UI.
- Cleaner tracking workflow for full possessions and lineups.
- Import/export bridge for the friend's React/Vite site.
